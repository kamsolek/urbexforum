# **University Project - Web Application**

This is a **university project** showcasing a complete **web application** built with **HTML, CSS, JavaScript, and PHP**. The project includes front-end styling, interactive elements, and backend functionality with a **database for user authentication and data storage**.

## **Key Features**

- **User Posts** - Each post includes a **title, description, and image**
- **Responsive Web Design** - Using `div`, `span`, CSS styling, and proper positioning
- **Login & Registration System** - User authentication with **password encryption**
- **Contact Form** - Basic form setup (initially without PHP processing)
- **JavaScript Functionalities:**
  - **Live Time Display**
  - **Email Validation**
  - **AJAX for Dynamic Content Updates**
- **Database Integration** - Storing form data and user credentials securely

This project demonstrates **full-stack web development** principles and serves as a foundation for more advanced applications.
